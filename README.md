# Labber-utilities
Utility functions for Labber
## QcodesToLabber.py
Requires the Labber API to run.
Contains two functions to:
1. Convert a single measurement from QCoDeS to a .hdf5 file
2. Convert all measurements from a QCoDeS database file (.db) to .hdf5 files
The appropriate folder structure for the Labber Log Browser is automatically generated
